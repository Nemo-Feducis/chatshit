How to Use the App

1) Every room must have a room code.

2) If you want to enter a room then ask the room owner for the room's code and room name.

3) If you want to create a room yourself, then name the room and set a room code for your room. Then share the room name and room code with your friends so you can chat with them.

4) The button besides the user's name is the user settings button, there you can change your name and profile picture.
